# v2 Test Infrastructure Plan

## Phase 1 — Unit Tests

Runner: Vitest
DOM: jsdom
Component: @testing-library/react
Coverage: @vitest/coverage-v8

Setup files:
- package/vitest.config.ts
- package/src/test/setup.ts

Target modules (priority order):
1. hooks/useNonNullableContext
2. Context reducers and dispatch logic
3. Button components (PlayBtn, PrevNnextBtn, RepeatTypeBtn, VolumeTriggerBtn, PlayListTriggerBtn)

## Phase 2 — E2E Tests

Runner: Playwright
Config: playwright.config.ts (repo root)
Test directory: e2e/

Target flows (priority order):
1. Player render and initialization
2. Play / Pause toggle
3. Track navigation
4. Volume and mute control
5. Repeat type cycling

## Phase 3 — GitHub Actions Pipeline

Workflow file: .github/workflows/test.yaml

Jobs:
- unit-test: runs vitest --coverage
- e2e-test: runs playwright (needs: unit-test)

Trigger: pull_request targeting v2/main, main, develop
